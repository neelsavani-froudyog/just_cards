## 4. Common Help Topics

## A. I scanned a card, but some details are incorrect
Business card scanning uses OCR and automated extraction. Sometimes text may be read incorrectly depending on:
- card quality,
- image angle,
- lighting,
- font style,
- multi-language text,
- handwritten notes.

You can manually edit saved contact details after scanning.

## B. A team member cannot see event contacts
Please check:
- the member was invited to the correct event,
- the invite was sent to the correct email address,
- the member accepted or joined successfully,
- the contacts were actually assigned to that event,
- the member has the correct permissions,
- the event is not private or restricted by organization settings.

## C. An organization member cannot access shared records
Please verify:
- the user is added to the organization,
- the event belongs to that organization,
- access roles are assigned correctly,
- the member is logged in with the correct account,
- there are no pending invite or access issues.

## D. Invite email is not received
Please ask the invited user to:
- check Spam, Junk, or Promotions folders,
- confirm the email address used for the invite,
- wait a few minutes and refresh,
- ensure email filters are not blocking the message.

If the issue continues, contact support and include:
- inviter email,
- invited email,
- event or organization name,
- approximate time the invite was sent.

## E. My uploaded file or PDF is not opening
Please check:
- file size limits,
- supported file type,
- internet connectivity,
- app permissions,
- whether the file has finished uploading.

If the problem continues, send us:
- file type,
- file size,
- contact record name,
- screenshot of the error.
